git config --global user.name "[nombre]"

Establece el nombre con el que hacer los commits

git config user.email "[email]"

Establece el email con el que hacer los commits

git config --global fetch.prune true

Limpia los origenes de las ramas locales que ya estan mergeadas al obtener

git config --global color.ui auto

Soporte para resaltar la sintaxis de consola

git config --global push.default upstream

Las ramas solo deberian hacer push a remotas trackeadas

git config --global alias.ci commit
git config --global alias.co checkout
git config --global alias.st status
git config --global alias.oops "commit --amend --no-edit"

Agregar al ultimo commit local (si no se ha hecho push) los ultimos cambios realizados

git config --global alias.untrack "rm --cache --"

Eliminar archivos de git, pero no del disco

git config --global alias.cls '!f() { git branch --merged ${1:-master} | egrep -v "(^\*|${1:-master})" | xargs --no-run-if-empty git branch -d; };f'

Asi como el fetch.prune limpia los origines de las ramas locales ya mergeadas, esto hace lo mismo directamente con las locales, excepto master

git reset [--hard] HEAD~N

Sustituyendo N por el numero de commits, deshace N commits locales y los devuelve a cambios pendientes. Si se usa la opcion --hard, se pierden los cambios.

git stash

Mete al stash los cambios pendientes

git stash pop|apply

Saca del stash los ultimos cambios guardados, en el caso de pop los elimina del stash si no hay conflictos

git stash drop

Elimina del stash los ultimos cambios guardados
